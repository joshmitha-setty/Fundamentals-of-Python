Write a python program to check whether the given character is vowel or consonant or Not an alphabet.
Sample Input: 
a 
Sample Output: 
Vowel

Solution:
s = input()
s = s.lower()
if s in "aeiou":
    print("Vowel")
elif s in "bcdfghjklmnpqrstvwxyz":
    print("Consonant")
else:
    print("Not an alphabet")
